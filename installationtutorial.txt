1. Install Oracle Database (im using Oracle 21c)
2. Install Xampp
3. Install oracle client
4. Open php.ini from apache config remove the first ';' from  this line ;extension=oci8_19  ; Use with Oracle Database 19 Instant Client
5. open SQLPlus 
6. Insert this table
CREATE TABLE account (
    account_id NUMBER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    username VARCHAR2(50) UNIQUE NOT NULL,
    password VARCHAR2(100) NOT NULL,
    mnemonic VARCHAR2(200) UNIQUE NOT NULL,
    created_at DATE DEFAULT SYSDATE
);
7. open cmd with htdocs directory and insert "git clone https://github.com/liamsimfty/PBL_Oron.git"
8. and open in browser localhost/pbl_oron/account.php
9. if table appear without any data open SQLPlus again insert 
INSERT INTO account (username, password, mnemonic) 
VALUES ('try', 'try', 'try');
10. and open again in browser localhost/pbl_oron/account.php if data still not exist open SQLPlus again insert GRANT "SELECT ON account TO system;"
11. the data will appear and if dont find yourself bitches

    username VARCHAR2(50) UNIQUE NOT NULL,
    kata sandi VARCHAR2(100) NOT NULL,
    mnemonic VARCHAR2(200) UNIQUE NOT NULL,
